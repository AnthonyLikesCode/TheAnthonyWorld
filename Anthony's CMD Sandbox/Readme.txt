Anthony's CMD Sandbox
Overview
Anthony's CMD Sandbox is an interactive application designed to empower users to create and execute Command Prompt (CMD) commands easily. It provides a user-friendly interface that allows both beginners and experienced coders to experiment with CMD scripting in a creative way.

Features
User-Friendly Interface: Intuitive design that simplifies command creation.
Custom Command Generation: Input simple phrases to generate corresponding CMD commands.
Interactive Sandbox: Experiment with different commands in a safe environment.
No Python Installation Required: Built as a standalone application for easy access.
Installation
Download the Executable: Obtain the latest version of the executable from the releases section.
Run the Application: Double-click the executable to launch Anthony's CMD Sandbox.
Usage
Open the application.
Input a command or phrase in the designated field (e.g., "Open settings").
Click the "Generate" button to create and execute the corresponding CMD command.
Explore the generated commands and modify them as desired.
