Anthony's Restart/Shutdown Assistant is a batch script designed to provide an easy-to-use interface for managing system shutdowns and restarts on Windows. It offers:

Administrative Verification: Ensures the script is run with the necessary administrative privileges.
User Authentication: Prompts for a specific name to access the main menu or an administrative menu.
Shutdown/Restart Scheduling: Allows you to schedule system shutdowns or restarts with delays specified in seconds, minutes, or hours.
Admin Functions: Access a special admin menu to view detailed system information.
